JavaScript 字符串方法详解

概述

JavaScript 字符串是不可变的，这意味着一旦创建，字符串的值就不能改变。对字符串进行操作的方法会返回新的字符串，而不会修改原始字符串。

1. 字符串拼接方法

+ 操作符

使用加号操作符可以连接两个或多个字符串。

示例：
let str1 = "Hello";
let str2 = "World";
let result = str1 + " " + str2;
console.log(result); // "Hello World"

+= 操作符

使用加等于操作符可以将一个字符串追加到另一个字符串。

示例：
let str = "Hello";
str += " World";
console.log(str); // "Hello World"

concat() 方法

concat() 方法用于连接两个或多个字符串，返回一个新的字符串。

示例：
let str1 = "Hello";
let str2 = "World";
let result = str1.concat(" ", str2);
console.log(result); // "Hello World"

2. 字符串替换方法

replace() 方法

replace() 方法用于在字符串中用另一个值替换指定的值，只替换第一个匹配项。

示例：
let str = "Hello World";
let newStr = str.replace("World", "JavaScript");
console.log(newStr); // "Hello JavaScript"

replaceAll() 方法

replaceAll() 方法用于在字符串中用另一个值替换所有匹配的值。

示例：
let str = "Hello World World";
let newStr = str.replaceAll("World", "JavaScript");
console.log(newStr); // "Hello JavaScript JavaScript"

交互式示例

3. 字符串截取方法

slice() 方法

slice() 方法提取字符串的一部分，并返回新的字符串。接受两个参数：起始索引和结束索引（不包含）。

示例：
let str = "Hello World";
let newStr = str.slice(0, 5);
console.log(newStr); // "Hello"

substring() 方法

substring() 方法提取字符串中两个指定索引之间的字符。与 slice() 类似，但不能接受负数参数。

示例：
let str = "Hello World";
let newStr = str.substring(0, 5);
console.log(newStr); // "Hello"

substr() 方法

substr() 方法从指定索引开始提取指定数量的字符。第一个参数是起始索引，第二个参数是要提取的字符数。

示例：
let str = "Hello World";
let newStr = str.substr(0, 5);
console.log(newStr); // "Hello"

4. 大小写转换方法

toUpperCase() 方法

toUpperCase() 方法将字符串转换为大写。

示例：
let str = "Hello World";
let newStr = str.toUpperCase();
console.log(newStr); // "HELLO WORLD"

toLowerCase() 方法

toLowerCase() 方法将字符串转换为小写。

示例：
let str = "Hello World";
let newStr = str.toLowerCase();
console.log(newStr); // "hello world"

交互式示例

5. 分割与连接方法

split() 方法

split() 方法将字符串分割成字符串数组，分割符由参数指定。

示例：
let str = "Hello,World,JavaScript";
let arr = str.split(",");
console.log(arr); // ["Hello", "World", "JavaScript"]

join() 方法

join() 方法将数组元素连接成字符串，连接符由参数指定。

示例：
let arr = ["Hello", "World", "JavaScript"];
let str = arr.join(" ");
console.log(str); // "Hello World JavaScript"

交互式示例

6. 去除空格方法

trim() 方法

trim() 方法去除字符串两端的空白字符。

示例：
let str = "  Hello World  ";
let newStr = str.trim();
console.log(newStr); // "Hello World"

trimStart() 方法

trimStart() 方法去除字符串开头的空白字符。

示例：
let str = "  Hello World  ";
let newStr = str.trimStart();
console.log(newStr); // "Hello World  "

trimEnd() 方法

trimEnd() 方法去除字符串结尾的空白字符。

示例：
let str = "  Hello World  ";
let newStr = str.trimEnd();
console.log(newStr); // "  Hello World"

交互式示例

7. 字符访问方法

charAt() 方法

charAt() 方法返回指定索引位置的字符。

示例：
let str = "Hello World";
let char = str.charAt(0);
console.log(char); // "H"

索引访问

可以使用方括号表示法直接访问字符串中指定索引位置的字符。

示例：
let str = "Hello World";
let char = str[0];
console.log(char); // "H"

at() 方法

at() 方法接受一个整数值作为参数，并返回指定索引位置的字符，支持负索引。

示例：
let str = "Hello World";
let char = str.at(0);
console.log(char); // "H"

let lastChar = str.at(-1);
console.log(lastChar); // "d"

交互式示例

8. 字符串填充方法

padStart() 方法

padStart() 方法用另一个字符串填充当前字符串（如果需要的话），使结果字符串达到指定长度，从当前字符串的开头开始填充。

示例：
let str = "5";
let paddedStr = str.padStart(3, "0");
console.log(paddedStr); // "005"

padEnd() 方法

padEnd() 方法用另一个字符串填充当前字符串（如果需要的话），使结果字符串达到指定长度，从当前字符串的结尾开始填充。

示例：
let str = "5";
let paddedStr = str.padEnd(3, "0");
console.log(paddedStr); // "500"

交互式示例

9. 字符串搜索方法

indexOf() 方法

indexOf() 方法返回指定值在字符串中首次出现的位置，如果未找到则返回 -1。

示例：
let str = "Hello World";
let index = str.indexOf("World");
console.log(index); // 6

lastIndexOf() 方法

lastIndexOf() 方法返回指定值在字符串中最后一次出现的位置，如果未找到则返回 -1。

示例：
let str = "Hello World World";
let index = str.lastIndexOf("World");
console.log(index); // 12

includes() 方法

includes() 方法用于检查字符串是否包含指定的子字符串，返回 true 或 false。

示例：
let str = "Hello World";
let result = str.includes("World");
console.log(result); // true

startsWith() 方法

startsWith() 方法用于检查字符串是否以指定的子字符串开头，返回 true 或 false。

示例：
let str = "Hello World";
let result = str.startsWith("Hello");
console.log(result); // true

endsWith() 方法

endsWith() 方法用于检查字符串是否以指定的子字符串结尾，返回 true 或 false。

示例：
let str = "Hello World";
let result = str.endsWith("World");
console.log(result); // true

交互式示例

10. 正则表达式匹配方法

match() 方法

match() 方法检索字符串中与正则表达式匹配的结果，返回匹配结果数组或 null。

示例：
let str = "Hello World";
let result = str.match(/World/);
console.log(result); // ["World", index: 6, input: "Hello World", groups: undefined]

search() 方法

search() 方法检索字符串中与正则表达式匹配的子字符串，返回首次匹配项的索引，如果未找到则返回 -1。

示例：
let str = "Hello World";
let index = str.search(/World/);
console.log(index); // 6

matchAll() 方法

matchAll() 方法返回一个包含所有匹配正则表达式的结果及分组捕获的迭代器。

示例：
let str = "Hello World World";
let matches = [...str.matchAll(/World/g)];
console.log(matches.length); // 2

注意事项

字符串不可变性：JavaScript 中的字符串是不可变的，这意味着字符串一旦创建就不能被修改。所有字符串方法都会返回新字符串，而不会修改原始字符串。

兼容性：某些较新的方法如 replaceAll()、at()、matchAll()、trimStart()、trimEnd()、padStart() 和 padEnd() 在较老的浏览器中可能不被支持。

性能：对于大量字符串操作，应该考虑性能影响。例如，频繁使用 + 或 += 操作符连接大量字符串可能不如使用数组 join() 方法高效。
