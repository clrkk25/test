# 项目主页

这个项目是一个静态网站，包含了各种前端开发示例和课程项目。

## 项目结构

```
.
├── BASIC/               # 基础示例目录
│   ├── 基础布局示例.html
│   ├── ES6特性示例.html
│   ├── flex布局演示.html
│   ├── 网格布局演示.html
│   ├── React虚拟DOM详解.html
│   ├── 视频播放器.html
│   └── 乒乓球游戏/
├── FCC/                 # FCC课程项目目录
│   ├── CSS/             # CSS项目
│   ├── HTML/            # HTML项目
│   ├── JS/              # JavaScript项目
│   └── REACT/           # React项目
├── src/                 # 资源文件目录
│   ├── audio/           # 音频文件
│   └── meida/           # 媒体文件
├── test/                # 测试文件目录
├── index.html           # 首页
├── styles.css           # 样式文件
├── script.js            # JavaScript文件
└── README.md            # 项目说明文件
```

## 技术栈

- HTML5
- CSS3
- JavaScript ES6+
