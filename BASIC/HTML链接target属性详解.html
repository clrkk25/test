HTML链接target属性详解

什么是target属性？

target属性用于指定链接在何处显示链接的资源。它定义了链接内容在浏览器中的打开方式。

target属性可以应用于<a>、<form>等元素。

target属性值及使用场景

target值 行为描述 使用场景 示例代码
_self 在当前窗口/标签页打开（默认值） 网站内部链接，页面跳转 <a href="page.html" target="_self">链接</a>
_blank 在新窗口/标签页打开 外部链接，希望保留当前页面 <a href="https://example.com" target="_blank">链接</a>
_parent 在父框架中打开 框架集布局中使用 <a href="page.html" target="_parent">链接</a>
_top 在顶层窗口打开 跳出框架集时使用 <a href="page.html" target="_top">链接</a>

详细示例

1. _self（默认行为）

在当前窗口/标签页中打开链接，这是默认行为：

在当前窗口打开视频播放器

特点：

  • 当前页面会被新页面替换
  • 浏览器后退按钮可以返回原页面
  • 不创建新窗口或标签页
  • 适用于网站内部导航
2. _blank（新窗口打开）

在新窗口或标签页中打开链接：

在新窗口打开视频播放器

特点：

  • 保留当前页面不变
  • 在新窗口/标签页中打开链接
  • 适用于外部链接
  • 用户可以同时查看两个页面

安全提醒

使用target="_blank"时，建议同时添加rel="noopener noreferrer"属性：

<a href="https://example.com" target="_blank" rel="noopener noreferrer">安全链接</a>

原因：

  • noopener：防止新页面通过window.opener访问原页面，提高安全性
  • noreferrer：不发送referrer信息，保护用户隐私

框架布局中的使用示例

在框架（frameset）布局中，target属性有特殊用途：

<frameset rows="20%,80%">
  <frame src="header.html" name="header">
  <frame src="content.html" name="content">
</frameset>

<!-- 在header框架中点击链接，在content框架中显示内容 -->
<a href="page.html" target="content">在内容框架中打开</a>

实际应用场景

网站导航

内部链接通常使用默认方式（_self）：

<nav>
  <a href="home.html">首页</a>
  <a href="about.html">关于我们</a>
  <a href="contact.html">联系我们</a>
</nav>

外部资源链接

外部链接建议使用_blank并添加安全属性：

<div class="external-links">
  <a href="https://www.example.com" target="_blank" rel="noopener noreferrer">访问示例网站</a>
  <a href="https://github.com" target="_blank" rel="noopener noreferrer">访问GitHub</a>
</div>

文档下载

文档下载链接通常在新窗口打开：

<a href="document.pdf" target="_blank" rel="noopener noreferrer">下载PDF文档</a>

了解target属性的不同值及其使用场景，可以帮助您更好地控制链接的打开方式，提升用户体验。
