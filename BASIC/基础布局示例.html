Flex布局示例 - 内容居中对齐

布局说明

Flex布局（弹性布局）是CSS3中一种新的布局模式，用于在容器中分配空间和对齐元素。

主要特点：

  • 适合一维布局（行或列）
  • 可以轻松实现元素的水平和垂直居中
  • 具有良好的灵活性和响应式特性
  • 通过justify-content和align-items属性控制对齐方式
/* Flex布局 */ .preview-box { display: flex; justify-content: center; align-items: center; background: linear-gradient(135deg, #3498db 0%, #2c3e50 100%); }
